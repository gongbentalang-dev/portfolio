# プロジェクト名
Photography Portfolio
## 概要
カメラユーザーが快適に写真を閲覧できることを目的としたWebサイトです。  
一覧から画像を拡大し、前後の画像へスムーズに移動できるUIを実装しています。  

また、JavaScriptによる動的処理の理解を目的として制作しました。

## 使用技術
・HTML / CSS / JavaScript  
・Lightbox  

## 主な機能
・スライドショー  
・画像拡大表示（Lightbox）  
・左右ナビゲーション  
・EXIF情報の表示  

## 工夫した点
・一覧に戻らず連続して閲覧できるUI設計  
・シンプルで直感的に操作できるデザイン  
・EXIF情報を表示し、撮影データも確認できるようにした点  

## 苦労した点
・Lightboxの実装におけるイベント管理  
・EXIF情報の取得と表示処理  

## 今後の課題

・UI/UXの改善

・画像のカテゴリ分け機能
